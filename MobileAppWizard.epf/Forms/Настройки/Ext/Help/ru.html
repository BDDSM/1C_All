В данной форме выполняется настройка параметров обработки.

  • Расположение мобильной платформы - каталог, в котором расположены файлы android.zip и ios.zip, необходимые для формирования поставки мобильного приложения. Если каталог не указан, то мобильная платформа ищется в каталоге 1C каталога документов текущего пользователя.
  • Каталоги установки SDK - содержат каталоги, в которых установлены Android SDK и JDK.
  • Размещение результата - указывает, в какие каталоги помещать результат работы обработки.
  • Параметры ключа для ОС Android - позволяет указать файл ключа, а также его представление и пароль. Данный параметры необходимы для осуществления подписывания файла мобильного приложения с приложением для работы на ОС Android.
  • Параметры нового ключа - если нужно создать новый ключ для подписи приложения, то это можно сделать, заполнив все поля в группе и нажав кнопку Сформировать файл ключа.

 

Android SDK можно получить по адресу: http://developer.android.com/sdk/index.html. Минимальные требования к Android SDK следующие:

  • Версия Android SDK Tools - не ниже 20.0.3;
  • Версия Android SDK Platform-tools - не ниже 14;
  • Версия SDK Platform - API 8 (не ниже версии 8.3).

Java SDK можно получить по адресу: http://www.oracle.com/technetwork/java/javase/downloads/jdk-6u25-download-346242.html (JDK7 не работает совместно с Android SDK).
